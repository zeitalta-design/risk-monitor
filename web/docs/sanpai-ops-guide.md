# sanpai 運用ガイド

## 概要
さんぱいくん（環境省系）から産業廃棄物処理業者の許可取消処分情報を取得し、
sanpai_items / sanpai_penalties に保存する。

## データソース
- **さんぱいくん全件検索**: `https://www2.sanpainet.or.jp/shobun/all_search.php`
  - 全国の許可取消処分情報 1,418件
  - テーブル形式: [処分日, 都道府県市, 事業者名, 許可番号]
  - 1ページに全件表示

## 推奨 cron 設定
```crontab
# 週次（月曜 7:00）— 推奨設定
0 7 * * 1 cd /path/to/web && SANPAI_MAX_ITEMS=500 node scripts/cron-sync.js --domain sanpai
```

## 環境変数
| 変数 | デフォルト | 推奨値 | 説明 |
|------|----------|-------|------|
| `SANPAI_MAX_ITEMS` | `100` | **500** | 1回あたりの最大取得件数 |

## 段階検証結果
| MAX_ITEMS | 取得件数 | 実行時間 | 安定性 |
|-----------|--------|--------|--------|
| 100 | 100件 | 0.5秒 | ✅ |
| 300 | 300件 | 0.9秒 | ✅ |
| **500** | **500件** | **0.6秒** | **✅ 推奨** |

## 実績（本運用）
- 実データ取得: **501件**（MAX_ITEMS=500）
- 実データ率: **98%**（501/511）
- 処分種別: 許可取消（さんぱいくんの掲載対象）
- 都道府県: **全国40+県**（埼玉40, 東京38, 千葉37, 神奈川24, 茨城24 等）
- penalties: 517件
- 複数penalty事業者: あり（最大4件）

## review条件
- 新規事業者: review_required
- 処分種別変化: review_required
- 既存データとの不整合: review_required
- 行政処分は誤判定コストが高いため、保守的に運用

## fallback ルール
| 条件 | 動作 |
|------|------|
| さんぱいくん取得成功 | 実データ優先 |
| さんぱいくん取得失敗 | fallback + エラー記録 |
| 東京都環境局取得成功 | 追加データとして取り込み |
